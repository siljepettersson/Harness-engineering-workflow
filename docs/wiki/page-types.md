# Wiki Page Types

Initial wiki page types:

- `concept`: terms such as CPI, HICP, CPI-ATE
- `method`: methodology, seasonal adjustment, weighting
- `comparison`: CPI vs HICP, CPI vs CPI-ATE
- `table`: Statbank table descriptions
- `analysis`: reviewed reusable answer pages

Use the page type to guide structure, source links, and lint checks.
