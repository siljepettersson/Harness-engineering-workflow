# Source Links

Source links are required because wiki pages are derived synthesis.

Rules:

- Definitions should link to curated raw SSB documentation under `data/raw/ssb/`.
- Original capture snapshots can be listed separately with `archive_links` when available.
- Comparison pages should cite sources for each compared concept.
- Numeric claims should link to structured data or table metadata.
- Wiki-to-wiki links do not replace raw source links.

The lint pipeline should flag pages without raw source grounding.
