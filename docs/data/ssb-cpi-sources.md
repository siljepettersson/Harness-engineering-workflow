# SSB CPI Sources

The first source set should stay small and curated.

Target source categories:

- CPI definition and overview
- CPI methodology
- HICP explanation
- seasonal adjustment explanation
- relevant Statbank table metadata

Use `data/raw/archive/ssb/cpi/` for original capture snapshots when practical, then create curated Markdown under `data/raw/ssb/cpi/` for retrieval and grounding.

Do not expand to other SSB topics until CPI is stable.
