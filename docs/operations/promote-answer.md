# Promote Answer To Wiki

Use this workflow when an answer is worth preserving as reusable synthesis.

1. Review the answer manually.
2. Verify claims against raw SSB sources or structured table output.
3. Create a page under `wiki/analysis/`.
4. Add frontmatter, source links, and related pages.
5. Update `wiki/index.md`.
6. Append an entry to `wiki/log.md`.
7. Run wiki lint when available.

Do not promote unreviewed or ungrounded answers.
