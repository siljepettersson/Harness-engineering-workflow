# Ingest Source

Use this workflow when adding a new SSB source.

1. Save the original capture snapshot under `data/raw/archive/ssb/cpi/` when practical.
2. Save the curated Markdown source under `data/raw/ssb/cpi/`.
3. Preserve source URL, retrieval date, license metadata, and archive path in frontmatter when available.
4. Update or create relevant wiki pages.
5. Update `wiki/index.md`.
6. Append an entry to `wiki/log.md`.
7. Run wiki lint when available.

Raw source content should not be edited to make an answer easier.
